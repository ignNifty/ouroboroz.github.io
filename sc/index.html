How to Use

Solar Client offers both an unblocker, and a game boy color emulator. To use the unblocker, go to www.solarclient.tk/textbooks/google.com (or any other link in place of google). The unblocker doesn't work for most things, but can be used to access minimal websites. The emulator can be found at www.solarclient.tk/gba most games work, but some had to be removed in order to control the file size of the website.
